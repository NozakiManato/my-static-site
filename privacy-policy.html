2. Information Collection and Use

2.1 Automatically Collected Information

The Application automatically collects the following information when you download, install, and use it:

  • Device Information: Operating system type and version, device model
  • Network Information: Your device's Internet Protocol (IP) address
  • Usage Data: Pages visited within the Application, time and date of visits, time spent on pages and in the Application

2.2 Location Information

⚠️ Background Location Tracking: This Application collects and uses precise location data continuously, even when the app is closed or not actively in use.

The Application collects your device's precise location for the following purposes:

  • Geofencing Services: Automatic detection when you enter or exit the designated university area (Kyushu Institute of Technology campus: latitude 33.6038, longitude 130.6952, radius 200 meters)
  • Background Monitoring: Continuous monitoring while your device is within or near the geofenced area, operating even when the app is not open
  • Periodic Verification: Background checks performed every 15 minutes while inside the monitored area to verify continued presence
  • Attendance Logging: Automatic recording of entry and exit timestamps for attendance verification

2.3 Bluetooth Information

The Application uses Bluetooth Low Energy (BLE) technology to:

  • Beacon Scanning: Continuously scan for nearby BLE beacons inside classroom areas
  • Device Connection: Connect to specific attendance verification devices using Service UUID 0000180a-0000-1000-8000-00805f9b34fb
  • Presence Confirmation: Verify your physical presence within designated classroom locations
  • Background Operation: BLE scanning continues in the background when you are inside the geofenced area

2.4 User Identification

The Application collects and processes:

  • User ID: A unique identifier associated with your student or faculty account
  • Attendance Records: Entry and exit timestamps, location status, and BLE connection events

2.5 Local Storage

The Application stores the following information on your device:

  • App state (OUTSIDE_AREA, INSIDE_AREA, UNCONFIRMED, CONFIRMED)
  • Temporary attendance data before server transmission
  • BLE connection history and status

3. Automated Data Processing

Important: This Application performs automated attendance recording without requiring manual user confirmation:

  • Automatic Entry Recording: When you enter the geofenced area, the app automatically sends an entry notification to our servers
  • Automatic Exit Recording: When you leave the geofenced area, the app automatically sends an exit notification
  • Automatic BLE Scanning: The app periodically initiates BLE scans in the background to verify classroom presence
  • Automatic Status Updates: App state changes trigger automatic API calls without user interaction

4. Data Transmission and Storage

4.1 Server Communication

Your attendance data is transmitted to our secure server:

  • Server Location: kyutech-4lab.jp (hosted by Render.com)
  • Data Transmitted: User ID, entry/exit timestamps, location status, BLE connection events
  • Transmission Frequency:
    • Upon entering the geofenced area
    • Upon exiting the geofenced area
    • During periodic background verification checks (every 15 minutes)
    • Upon BLE beacon connection/disconnection
  • Protocol: HTTPS encrypted connections

4.2 API Endpoints

The Application communicates with the following endpoints:

  • POST https://kyutech-a.onrender.com/api/attendance/enter - Entry recording
  • POST https://kyutech-a.onrender.com/api/attendance/exit - Exit recording

8. Your Rights and Choices

8.1 Opt-Out and Location Control

You can control or stop location data collection through your device settings:

iOS:

  1. Open Settings → Privacy & Security → Location Services
  2. Find "Kyutech BLE" and select:
    • Never: Completely disable location tracking
    • While Using the App: Enable only when app is open (disables background tracking)
    • Always: Full functionality with background tracking

Android:

  1. Open Settings → Apps → Kyutech BLE → Permissions
  2. Tap "Location" and select:
    • Don't allow: Completely disable location tracking
    • Allow only while using the app: Enable only when app is open
    • Allow all the time: Full functionality with background tracking

⚠️ Important: Disabling background location or Bluetooth will prevent automatic attendance tracking. You may not receive attendance credit without these permissions enabled.

8.2 Complete Opt-Out

You can stop all data collection by uninstalling the Application using your device's standard uninstall process.
